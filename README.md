### 🛠 Setting-up

```
git clone https://github.com/Plasmonix/Windscribe-Checker
cd Windscribe-Checker
pip install -r requirements.txt
python main.py
```
---------------------------------------

### 🏆 Features 
- Ultra fast checking.
- User friendly UI.
- Multithreaded. 
- Proxy support (HTTPS, SOCKS4, SOCKS5).
- High CPM.
- Save hits to file.
